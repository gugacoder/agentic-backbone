**Assistant:** A ferramenta `gitlab_issues` **não está disponível** nesta sessão. Ela não consta na lista de ferramentas carregadas nem nas ferramentas diferidas.

**Alternativas para listar issues do GitLab:**

1. **Via CLI `glab`** (se instalado):
   ```bash
   glab issue list --opened
   ```

2. **Via API REST do GitLab**:
   ```bash
   curl --header "PRIVATE-TOKEN: <seu-token>" \
     "https://gitlab.com/api/v4/projects/<project-id>/issues?state=opened"
   ```

Quer que eu tente alguma dessas alternativas? Se sim, preciso saber:
- A **URL do seu GitLab** (ex: `gitlab.com` ou self-hosted)
- O **ID ou path do projeto**
- Um **token de acesso** (Personal Access Token)
