Registra uma nota manual no daily log de hoje (`kb/calendar/notes/YYYY-MM-DD.md`).

O argumento é o conteúdo da nota. Se nenhum argumento for fornecido, peça ao usuário o que ele quer registrar.

Passos:
1. Se `kb/calendar/notes/YYYY-MM-DD.md` não existir, crie com o header `# Daily Log: YYYY-MM-DD\n\n## Sessions\n\n## Memory Maintenance\n\n`
2. Faça append da nota como `### Note (HH:MM)\n\n<conteúdo>\n\n`
3. Confirme que a nota foi salva
